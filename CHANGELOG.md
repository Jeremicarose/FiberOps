## Release Notes Format

- Use `v0.x.y` tags for releases and `v0.x.y-rc.N` for prereleases.
- Add the release date in `YYYY-MM-DD` format.
- Group notable changes under `Added`, `Changed`, `Fixed`, and `Docs` when useful.
