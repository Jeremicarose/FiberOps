# Code Of Conduct

This repository is for technical collaboration. Everyone participating in issues, pull requests, and reviews is expected to keep the conversation respectful and useful.

## Expected Behavior

- Critique ideas, code, and decisions without attacking people.
- Assume good intent, then ask for clarification when something is unclear.
- Keep feedback specific, factual, and actionable.
- Respect different experience levels and explain jargon when it matters.
- Share security-sensitive information privately when possible.

## Unacceptable Behavior

- Harassment, discrimination, or intimidation
- Personal attacks or repeated bad-faith escalation
- Doxxing or sharing private credentials, tokens, or exploit details publicly
- Spam, trolling, or intentionally disruptive behavior

## Scope

This applies to repository discussions, issues, pull requests, review comments, and other project communication channels maintained for FiberOps.

## Reporting

If you encounter behavior that violates this code of conduct, contact the maintainer privately through the repository profile or an available private security channel. Include links or screenshots when possible.

## Enforcement

The maintainer may edit, hide, lock, or remove content that violates these expectations and may restrict repeat offenders from participating.
